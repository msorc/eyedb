This folder contains library files, which you can use in your projects.

ant.jar ant build tool, used to build db4o sources (http://ant.apache.org/)
bloat-1.0.jar bytecode optimization library, used to optmize Native Queries and Transparent Activation
db4o-7.2.54.11278-bench.jar IO benchmark library
db4o-7.2.54.11278-instrumentation.jar Instrumentation layer on top of bloat
db4o-7.2.54.11278-db4ounit.jar testing framework for db4o needs
db4o-7.2.54.11278-java1.1.jar db4o database engine core for JDK1.1
db4o-7.2.54.11278-java1.2.jar db4o database engine core for JDK 1.2 - JDK 1.4
db4o-7.2.54.11278-java5.jar db4o database engine core for JDK 5 and JDK 6
db4o-7.2.54.11278-nqopt.jar Native Query optimization library. This library should be available in the classpath for NQ optimization
db4o-7.2.54.11278-osgi.jar db4o for OSGi. See OSGi API documentation.
db4o-7.2.54.11278-osgi-test.jar db4o OSGI tests
db4o-7.2.54.11278-taj.jar db4o instrumentation classes for Transparent Activation
db4o-7.2.54.11278-tools.jar db4o tools user interface, contains db4o enhancer and Ant tasks


USING DB4O LIBRARIES

For using db4o with JDK1.1 you will only need db4o-7.2.54.11278-java1.1.jar.

For JDK 1.2 - JDK 6 you will need the respective db4o core database engine library.

For Native Query optimization on JDK 1.2 - JDK 6 you will additionally need:
bloat-1.0.jar
db4o-7.2.54.11278-classedit.jar
db4o-nqopt-7.2.54.11278.jar
db4o-7.2.54.11278-tools.jar

For Transparent Activation optimization on JDK 1.2 - JDK 6 you will additionally need:
bloat-1.0.jar
db4o-7.2.54.11278-classedit.jar
db4o-7.2.54.11278-taj.jar
db4o-7.2.54.11278-tools.jar
